## What is MDX?

MDX allows you to use JSX in your markdown content. This means you can import and use React components directly in your blog posts.

## Getting Started

To create a new blog post:

1. Create a new `.mdx` file in `src/features/blog/content/`
2. Add frontmatter with `title`, `publishedAt`, and `summary`
3. Write your content using Markdown syntax
4. Build and deploy!
